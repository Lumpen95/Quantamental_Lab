# Quantamental_Lab
시장에 대해 항상 겸손해라. 바다 속 물고기들처럼 물결을 따라 헤엄치면서, 물결을 거스르다가 좌절하지 않도록 노력하는 것이 최선이다.


### 단기 목표
1. 추세추종, 차트 관련 책 읽기
2. 가격, 거래량 기반 시장 regime 모델 개발
3. 리스크관리 규칙 어느정도 익히면 손절 및 익절라인 자동 계산 모듈 그림 그리기

### 매매 규칙
**1 추세가 살아있는가. 살아있다면 어느 단계인가(STAGE 1 추세 시작, STAGE 2 조정, STAGE 3 돌파)**
1) 지수 대비 RS강도 파악
2) ADX 지표
3) MACD 히스토그램 - 매수/매도 타이밍 분석
- 주가 상승/HIST 상승(0 이상): 강력한 상승 추세. 홀딩
- 주가 상승/HIST 횡보(0 이상): 홀딩
- 주가 상승/HIST 하락(0 이상): 매도세력이 강해지며 모멘텀 약화 인지.

- 주가 횡보/HIST 상승: 내부적으로 모멘텀이 축적 중으로 STAGE 3 추세 돌파 종목. 돌파 매매 준비
- 주가 횡보/HIST 하락: 관망하며 HIST 추세가 상승으로 전환되는 시기에 매수 시도. 하락이 멈추기 전에는 매수 금지

- 주가 하락/HIST 상승(0 이하): 하락 추세의 종료가 임박했으며 매수 세력이 점점 강해지고 있음. 반등 가능성 탐색
반복적인 HIST 저점 상승이 관측되어야 함
- 주가 하락/HIST 하락(0 이하): 매수 금지

4) 거래량
대량 거래는 일반적으로 추세 시작과 끝 시점에 나타난다
개별 주식의 유동 주식 비율 계산(플로트 비율): 50% 이상
플로트 비율 대비 일간 거래량 계산
일반적으로 상승 초기 추세가 지속되는 주식은 높은 일간 거래량을 나타낸다
주가가 상승하지만 거래량이 감소한다면 추세 전환 가능성 높음
추세선, 이평선을 하향 돌파하면서 높은 거래량을 보인다면 뚜렷한 하락 추세 전환 신호
